Mass Serial Number

Helping you assign mass serial number for your Manufacturing application

How can I assign Serial Number for 500+ products that are going to be produced?

We offer you a simple step in Odoo Manufacturing application which can help you assign mass serial number for your production order

We add a new button "Mass Produce" Manufacturing Order form

When clicking, a wizard will show up to help you with assigning serial number

Two ways to generate serial numbers

Offering you 2 ways to generate serial number depending on your demand

You need to enter your first serial number to Scan Lot as a template, then choose 1 of 2 ways to generate the rest:

Next Qty: This will help you generate amount of qty serial number start from Scan Lot

To Serial Number will help you generate serial number start from Scan Lot to "To Serial Number"

Usage Note

Some notes when using this module

Choose only 1 SN generating type

Just leave the other type with 0, if you change value for both 2 type, "Next Qty" will be run.

Importing your own Serial Number file

If you have a Serial Number file and want to import, please choose "Up load your file" and import it. The file should only have the list of serial number in 1 column.

Duplicated Serial Number Free

When importing, if a duplicated SN is found, file cannot be imported. We also prevent generating duplicated serial
